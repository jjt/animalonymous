animalonymous

An npm module that hashes input strings to output strings of the form "[adjective] [animal]". Intended to provide consistent (semi)-anonymous usernames.

Example

 

Install

$> npm install --save animalonymous

Usage

var input = 'test@animalonymous.com',
    output = require('animalonymous')(input);

console.log("Input: " + input + ", Output: " + output);
// -> "Input: test@animalonymous.com, Output: Upset Camel"

Details

We have two source arrays comprised of 591 animals and 1344 adjectives which get combined for a total of 794,304 unique outputs. The main function in animalonymous takes an md5 hash of the input, then converts the first 8 bytes to a 32-bit integer for selecting the animals, and then the next 8 bytes of the hash to a 32-bit integer for the adjectives. It then takes these integers and determines indexes for each array by floor( (int / 2^32) * len ). It then returns adjectives[adjectivesIndex] + " " + animals[animalsIndex].
